# MELS Standard — NFT MELS v1.1

MELS (Mesh Embedded Logo Signature) is an open technical standard for 3D NFTs.  
It defines how a creator’s 3D logo is embedded directly into the mesh of an artwork, enabling identity anchoring, traceability, and AI‑based verification.

This repository contains:

- The full MELS v1.1 specification (English)
- A short summary version
- Version history
- Legal notices

## Status

- MELS v1.0 → Method (logo embedded in the mesh)
- MELS v1.1 → NFT MELS (identity + traceability)
